🛡️ 개인정보 처리방침

최종 업데이트: 2024년 9월 2일

요약: Valorant Analytics는 사용자의 개인정보를 최소한으로 수집하며, 수집된 정보는 서비스 제공 목적으로만 사용됩니다.

1. 수집하는 정보

1.1 필수 정보

  • Riot ID: 게임 데이터 조회를 위한 플레이어 식별자
  • 게임 데이터: Riot Games API를 통해 제공되는 공개 매치 정보

1.2 자동 수집 정보

  • 사용 통계: 앱 사용 패턴 (익명화)
  • 오류 로그: 서비스 개선을 위한 기술적 정보

1.3 수집하지 않는 정보

  • 실명, 주소, 전화번호 등 개인 식별 정보
  • 결제 정보 (무료 서비스)
  • 게임 외 개인 활동 정보

2. 정보 사용 목적

  • 서비스 제공: 매치 분석, 통계 제공
  • 서비스 개선: 기능 향상, 버그 수정
  • 사용자 지원: 문의사항 처리

3. 정보 보관 및 삭제

3.1 보관 기간

  • 캐시 데이터: 최대 5분 (성능 향상 목적)
  • 사용 통계: 1년 (익명화된 데이터)
  • 오류 로그: 30일

3.2 데이터 삭제

사용자는 언제든지 다음 방법으로 데이터 삭제를 요청할 수 있습니다:

  • 앱 내 "데이터 삭제" 기능 사용
  • 개발자에게 직접 연락

4. 정보 보안

  • 암호화: 모든 데이터 전송 시 HTTPS 사용
  • 접근 제한: 개발자만 기술적 데이터 접근 가능
  • 로컬 처리: 가능한 모든 분석을 사용자 기기에서 수행

5. 제3자 공유

중요: 사용자의 개인정보를 제3자와 공유하지 않습니다.

5.1 예외 상황

다음의 경우에만 법적 요구에 따라 정보를 제공할 수 있습니다:

  • 법원의 명령
  • 수사기관의 요청 (적법한 절차에 따라)

6. Riot Games API 이용

본 서비스는 Riot Games API를 사용합니다:

  • 데이터 출처: Riot Games의 공식 API
  • 데이터 범위: 공개적으로 제공되는 게임 통계만
  • Riot 정책 준수: Riot Games의 API 이용약관 완전 준수

7. 사용자 권리

7.1 정보 열람권

사용자는 자신의 정보 처리 현황을 언제든지 확인할 수 있습니다.

7.2 정정·삭제권

잘못된 정보의 수정이나 삭제를 요청할 수 있습니다.

7.3 처리정지권

개인정보 처리 중단을 요청할 수 있습니다.

개인정보 처리방침 변경 시:

  • 앱 내 공지 (30일 전)
  • 웹사이트 게시
  • 중요 변경사항은 개별 통지
